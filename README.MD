## **How to run pytest:**
 - For short information (usually use for smoke test): 
    - pytest --tb no -m smoke
    - pytest --tb line -m smoke
 - For long information: 
    - pytest --tb short -m smoke
 - For full information: 
    - pytest -m smoke


## **Test module name *(case sensitive)*:**
- **payment**: Test case for payment
- **test**: testing purpose
